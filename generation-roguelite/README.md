# generation-roguelite

Godot 4.6 (.NET/C#) で開発する「世代継承ローグライト」のプロトタイプです。

## 現在の実装範囲（Phase 1〜9）

- 自動前進を前提にした障害物生成と回避判定
- 年齢進行（時間経過で加齢）
- 人生フェーズ切り替え（幼少期 / 青年期 / 壮年期 / 老年期）
- テンプレートイベント（SLM連携前の仮実装）
- タップ / スワイプ入力でのアクション・イベント選択
- 寿命システム（障害物失敗やイベント失敗で寿命減少）
- 世代交代（死亡 → 弔いメッセージ → 次世代開始）
- 遺伝・遺産の基本引き継ぎ
- SLM連携の基盤実装（PromptBuilder / PInvoke Bridge / JSON Parser / Validator）
- イベント事前キャッシュ（20〜30想定、現状Target=24）
- 地形パラメータ連動（event terrain → 障害物密度/速度へ反映）
- DDA（直近30秒回避率による障害物密度±0.2補正）
- ナビゲーター発話（イベント要約・成功失敗リアクション・フェーズ遷移・雑談）
- ナビ音声再生経路（OGG配置時に自動再生、未配置時は無音フォールバック）
- 家訓システム（世代スコアでポイント獲得→家訓自動制定）
- 縁システム（NPC親密度の蓄積とイベント判定支援）
- 時代変化（3世代ごと遷移）+ 時代固有メカニクス補正
- 災厄 / 黄金期（世代条件で難易度・スコア倍率が変動）
- ハクスラ（イベント成功時中心の装備ドロップ、レアリティ、家宝継承）
- 家系図記録（世代ごとの年齢・死因・時代・スコアの蓄積）
- 死因コレクション / 称号コレクション
- 実績システム（条件達成で称号解除）
- 始祖システム（進行状況で始祖タイプを段階解除）
- 家系の伝説（10世代ごとの自動生成）
- 後悔イベント（老年期に過去を振り返る演出）
- 強制広告 / リワード広告の管理（広告除去購入で強制広告OFF）
- IAPモック（広告除去、プレミアムパス、追加ナビキャラ購入）
- バトルパス（シーズン、レベル、無料/有料報酬、コスメ解放）
- Androidエクスポートプリセット追加（export_presets.cfg）
- パフォーマンス監視（FPS/メモリ）+ メモリ閾値時キャッシュトリム
- セーブ/ロード（IAP・バトルパス・コスメ・非同期共有状態）
- 追加ナビキャラ切替（mentor/trickster/oracle）
- 季節コスメ更新（シーズン進行でテーマ追加）
- 非同期マルチ基盤（ローカルランキング・伝説共有キュー）
- 世界観拡張（原典世界→異世界編→SF編）
- 段階的インフレ調整（世代/総スコア/時代/拡張テーマ連動）
- 時代別スコア単位テーブル + インフレ係数テーブル（Phase 8）
- 非同期縁談支援（順位連動ボーナスキュー + 次世代適用 + 永続化）（Phase 9）
- ローカルランキングの永続化（セーブ/ロード対応）（Phase 9）
- 弔い演出オーバーレイ（フェードアウト→弔い画面→次世代選択）
- 次世代生成UI（実子/養子選択、遺産プレビュー、誕生時広告ボーナスプレースホルダー）
- 世代履歴データ（GenerationHistory）と世代連続倍率による累計スコア算出
- 壮年期の伴侶候補イベント（魅力判定）と老年期突入時の養子強制フラグ

## 起動方法

1. Godot .NET 4.6 を起動
2. 本ディレクトリをプロジェクトとして読み込み
3. `Scenes/Main.tscn` をメインシーンとして実行

指定済み Godot パス:

- `D:\code_workspace\Godot_v4.6.1-stable_mono_win64\Godot_v4.6.1-stable_mono_win64.exe`

## デバッグ操作（PC）

- `Space` : タップ操作
- `S` : スワイプ操作

モバイル相当では、短いタッチをタップ、一定距離のドラッグをスワイプとして判定します。

### プロトタイプ仕様（docs/game_system.md）準拠メモ

- ゲーム内時間は「リアル1秒 = ゲーム内1年」
- イベントは 15〜30秒間隔で表示、制限時間は 8秒固定
- イベント選択は `A/B/C` の3ボタン（時間切れ時は低リスク選択肢を自動選択）
- 死亡時は即次世代へ進まず一時停止し、タップ/スワイプで次世代開始

### デバッグ用ショートカット（Phase 6/7検証）

- `F5` : 広告除去購入
- `F6` : プレミアムパス購入
- `F7` : 追加ナビキャラ購入（順次）
- `F8` : ナビキャラ切替
- `F9` : 継承ボーナス広告
- `F10` : イベントやり直し広告
- `F11` : ショップ枠広告
