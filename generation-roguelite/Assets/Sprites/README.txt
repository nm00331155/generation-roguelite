=== Sprite Sheet Placement Guide ===

Place the following 6 files in this folder:

  female_idle.png   - Static standing (sheet 4)
  female_run.png    - Running cycle (sheet 5)
  female_move.png   - Sliding move (sheet 6, rename from female_dodge.png)
  female_guard.png  - Crouch guard (sheet 2)
  female_hurt.png   - Hit reaction (sheet 3)
  female_death.png  - Collapse (sheet 1)

All sheets: 1024x1024, 3x3 grid, 9 frames each
Frame size: 341x341

After placing, open Godot and create SpriteFrames resource.
